Cancer Prediction ML

(a) Overview
Machine Learning model to predict whether a tumor is malignant or benign using medical features

(b)Tech Stack
1 Python
2 Pandas NumPy
3 Scikit learn
4 Seaborn

(c) Workflow
1 Data cleaning and preprocessing
2 Feature scaling using StandardScaler
3 Train test split 70 30
4 Logistic regression model
5 Evaluation using accuracy and classification report

(d) Result
Achieved about 98 percent accuracy in tumor classification

(e) Run Locally
1 git clone https://github.com/your-username/cancer-prediction.git
2 cd cancer-prediction
3 pip install pandas numpy scikit-learn seaborn
4 jupyter notebook
